- reference : [빅스스타](https://www.youtube.com/watch?v=IiGRhhhjAS8)

## 주제1 : **아프리카 코로나 확진자**
  - 아프리카 국가별 코로나 확진자 대시보드
    - 막대그래프
    - 그룹 설정
    - 색상 인코딩
  - 아프리카 월별 확진자 평균 추이
    - 라인 차트 만들기
    - 평균선, 추세선 추가하기
  - 아프리카 국가별 확진자수 맵
